Tentang Saya

Saya merupakan seorang sarjana dari jurusan Sistem Informasi di Universitas Sriwijaya tahun 2018 dengan IPK 3.51. Dengan latar belakang pendidikan tersebut, saya memiliki kemampuan dalam menganalisa dan merancang sebuah sistem, serta mengimplementasikannya menjadi sebuah sistem yang dapat dijalankan sesuai kebutuhan. Dari sejak berkuliah hingga sekarang, secara mandiri saya menjadi freelancer di bidang pemrograman web yang fokus menggunakan bahasa pemrograman PHP, framework Laravel, Framework CodeIgniter dan menggunakan MySQL/PostgreSQL sebagai database servernya. Karena ketertarikan saya terhadap bidang pemgrogaman membuat saya selalu termotivasi untuk memperdalam ilmu dan terus belajar dalam bidang ini.
